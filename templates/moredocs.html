How to use the this website to test the API's


1. Generate token

To generate token click on in /api/auth/token/ and provide username and passoword


2. Register

To register click on in /api/users/register/ and provide username,email and passoword


3. Create Article

To Create Article click on in /api/article/create/ and provide title,body and token in header for authorization. Token value should be preceeded by "JWT" as shown below.


4. View/Retrieve Article

To View Article click on in /api/article/ and press send.


5. Create Comment

To Create Comment click on in /api/article/<pk>/comment/create and provide text,article primary key value and token in header for authorization. Token value should be preceeded by "JWT" as shown below. Also <pk> in URL endpoint should be replaced by article primary key value.


6. View/Retrieve Comment

To View Comment click on in /api/article/<pk>/comment/ and replace URL endpoint by article primary key value
